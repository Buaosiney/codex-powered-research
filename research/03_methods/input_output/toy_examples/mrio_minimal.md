# MRIO 最小数值例子

## 目标

用一个“两地区、每地区一个部门”的极简例子理解：

- 什么是多区域投入产出分析
- 为什么一个地区的最终需求会带动另一个地区的生产

## 场景设定

假设只有两个地区：

1. 地区 A
2. 地区 B

每个地区只有一个综合部门。

直接消耗系数矩阵设为：

```text
A = | 0.20  0.10 |
    | 0.15  0.25 |
```

解释：

- A 地区每生产 1 单位，需要：
  - A 地区自身投入 0.20
  - B 地区投入 0.15

- B 地区每生产 1 单位，需要：
  - A 地区投入 0.10
  - B 地区自身投入 0.25

现在设最终需求为：

```text
y = | 100 |
    |   0 |
```

表示只有地区 A 有 100 单位最终需求，地区 B 没有最终需求。

## 第一步：构造 Leontief 逆矩阵

```text
I - A = | 0.80  -0.10 |
        | -0.15  0.75 |
```

求逆后：

```text
L = (I - A)^(-1) ≈ | 1.2903  0.1720 |
                   | 0.2581  1.3763 |
```

## 第二步：计算总产出

```text
x = Ly
```

代入：

```text
x ≈ | 129.03 |
    |  25.81 |
```

## 如何解释结果

结果表明：

- 虽然最终需求只发生在地区 A
- 但为了满足 A 的需求，地区 B 也需要生产约 `25.81`

这就是 MRIO 的核心：

`一个地区的需求，会通过跨区域产业链激活其他地区的生产。`

## 如果再加入环境强度

假设：

```text
e = | 0.4  0.9 |
```

表示：

- A 地区单位产出排放强度为 0.4
- B 地区单位产出排放强度为 0.9

则总排放为：

```text
f = ex
  = 0.4 × 129.03 + 0.9 × 25.81
  ≈ 74.85
```

## 关键直觉

1. 地区 A 的需求不只影响 A
2. 上游依赖会把环境压力转移到其他地区
3. 这正是全球消费、贸易和投资环境影响分析的基础逻辑

## 与你的主线的关系

你的导师方向强调：

- 消费、贸易与投资活动在全球的资源环境影响

MRIO 正好是这类问题的核心语言。  
以后如果你研究：

- 基础设施投资是否依赖高排放国际上游产业链
- 固定资本形成的环境影响是否跨国转移

本质上都要先掌握这种跨区域传导直觉。
