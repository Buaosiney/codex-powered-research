# 环境扩展投入产出最小数值例子

## 目标

在 Leontief 逆矩阵例子的基础上，再加入一个排放强度向量，理解：

- 为什么最终需求会对应总排放
- 为什么不能只看直接排放

## 已知条件

沿用前一个例子：

```text
L ≈ | 1.3115  0.1639 |
    | 0.4918  1.3115 |
```

最终需求：

```text
y = | 100 |
    |  50 |
```

所以总产出：

```text
x = Ly ≈ | 139.34 |
         | 114.75 |
```

## 加入环境强度

假设两个部门的单位产出排放强度为：

```text
e = | 0.5  0.8 |
```

表示：

- 能源部门每单位产出排放 `0.5`
- 制造部门每单位产出排放 `0.8`

## 计算总排放

环境扩展 IO 的核心表达为：

```text
f = eLy
```

也可以分两步理解：

1. 先算总产出 `x = Ly`
2. 再算总排放 `f = ex`

代入数值：

```text
f = 0.5 × 139.34 + 0.8 × 114.75
  ≈ 69.67 + 91.80
  ≈ 161.47
```

## 如何解释结果

这说明：

- 为满足这组最终需求
- 整个经济系统中被激活的直接和间接生产过程
- 总共对应约 `161.47` 单位排放

## 为什么不能只看直接排放

如果只看最终需求对应部门的“表面排放”，你会漏掉很多上游过程。

例如：

- 制造最终品的背后，还需要能源、材料和更多制造投入
- 这些上游过程也会产生排放

EEIO 的价值就在于把这些间接影响也纳入核算。

## 关键直觉

1. `L` 解决的是“总产出需求”
2. `e` 解决的是“每单位产出的环境压力”
3. `eLy` 把两者接起来，形成需求端的总环境影响核算

## 与你的主线的关系

这一步非常关键，因为你的研究不会停留在“产出变化”本身，而是会进入：

- 碳排放
- 能源使用
- 资源消耗
- 投资驱动的环境影响

未来你要研究固定资本形成和基础设施投资时，本质上也是在问：

`某类投资需求通过产业链激活了多少总环境压力？`
