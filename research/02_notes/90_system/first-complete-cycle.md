# 第一次完整流转说明

这份说明的目标是让你真正跑完一次最小工作流，而不是继续搭系统。

完整流转指的是：

`文献 -> 笔记 -> 方法卡片/问题笔记 -> 研究备忘录`

只要跑通一次，初始化阶段就基本结束。

## 推荐你明天做的最小流转对象

优先选这篇：

- [Environmental Impacts of Capital Formation](../01_literature_notes/lit-sodersten-2018-capital-formation.md)

原因：

- 直接贴近导师主线
- 与你的问题笔记已经接上
- 与后续资本形成研究高度相关

## 第一步：在 Zotero 中补入文献

目标：

- 让这篇文章真正进入文献事实库，而不是只存在于 Obsidian 里

最低要求：

- [ ] 条目录入
- [ ] PDF 或链接可找到
- [ ] 标签至少包括 `core`、`capital-formation`、`investment`

## 第二步：检查或补全文献笔记

打开：

- [lit-sodersten-2018-capital-formation.md](../01_literature_notes/lit-sodersten-2018-capital-formation.md)

你要做的不是重写，而是检查 3 件事：

1. 这篇文献的研究问题，你能不能用自己的话再说一遍
2. 这篇文献为什么和导师方向直接相关
3. 它最值得你借用的点是什么

如果有需要，只补 2-3 句，不要过度加工。

## 第三步：连接到方法卡片

至少连接下面两张：

- [环境扩展投入产出分析](../02_method_cards/method-environmentally-extended-input-output.md)
- [多区域投入产出分析](../02_method_cards/method-multi-regional-input-output.md)

你要回答的不是“它用了什么高深方法”，而是：

- 为什么这个问题需要用 EEIO / MRIO，而不是简单直接排放统计

## 第四步：连接到问题笔记

打开：

- [固定资本形成与长期排放影响](../03_question_notes/question-fixed-capital-and-emissions.md)

补一条你自己的判断，例如：

- 这篇文章证明固定资本形成不能被当成普通最终需求项简单处理
- 资本形成的结构差异可能是国家间碳足迹差异的重要来源

只写 1-2 条就够。

## 第五步：更新研究备忘录

打开：

- [为什么入学前先以投入产出分析为方法中轴](../05_research_memos/memo-2026-04-16-why-io-first.md)

补一小段即可，建议写：

- 这篇文章为什么强化了“IO 应先学”的判断
- 它暴露了什么后续方法升级方向

## 第六步：结束动作

这一步非常简单，但很重要。

完成后你要做两件事：

1. 建一份周回顾，哪怕只写 5 行
2. 把今天完成的内容 `git add -> git commit -> git push`

## 跑通之后你会得到什么

如果这轮做完，你就不再只是“有一个系统”，而是已经真正使用了它。

你将得到：

- 一篇进入 Zotero 的核心论文
- 一份与你方向直接相关的文献笔记
- 至少两张被激活的方法卡片
- 一条正在成形的问题链
- 一份开始积累判断的研究备忘录

这就是后续所有研究工作的最小原型。
