# 工作流初始化检查清单

## A. 今晚不做也没问题

以下项目不需要现在立刻完成：

- [ ] 下载大型数据库
- [ ] 配置 Obsidian 插件
- [ ] 美化目录或模板
- [ ] 做复杂自动化
- [ ] 开始系统补线性代数

这些全部可以留到后面。

## B. 明天应优先完成

### B1. Obsidian 接入

- [ ] 确认 `research/02_notes/` 作为 vault 使用
- [ ] 检查 `00_dashboard/`、`01_literature_notes/`、`02_method_cards/` 是否显示正常
- [ ] 确认 `99_templates/` 中模板可见

### B2. Zotero 初始同步

- [ ] 在 Zotero 中创建或确认与当前方向匹配的 collection / tag
- [ ] 补入首批核心论文
- [ ] 为每篇核心论文添加最少标签

### B3. 第一轮真实流转

- [ ] 选 1 篇论文
- [ ] 检查已有文献笔记或新建文献笔记
- [ ] 将该论文连接到至少 1 张方法卡片
- [ ] 将该论文连接到至少 1 条问题笔记
- [ ] 在研究备忘录中更新 2-3 句判断

### B4. 方法学习启动

- [ ] 阅读 `method-leontief-inverse.md`
- [ ] 阅读 `leontief_inverse_minimal.md`
- [ ] 手算或代码复现一次最小例子

## C. 初始化完成的判断标准

当以下条件都满足时，初始化阶段可以视为完成：

- [ ] 目录骨架已稳定使用
- [ ] Obsidian 已成为主知识入口
- [ ] Zotero 已接入首批核心论文
- [ ] 至少完成一次完整流转
- [ ] 至少完成一次最小数值例子练习

## D. 初始化后再做的事

- [ ] 下载 WIOD 或 OECD ICIO
- [ ] 写第一份周回顾
- [ ] 建立第一份真实项目 README
